Projects

I'm a firm believer in sharing knowledge and open source software. Because of that, most of my projects end up on Github for anyone to check out.

  • FT Night Radian LED Controller

    A joint project with my dad, we designed and coded a replacement LED controller for the FT Night Radian. The LED patterns are customizable, and can be controlled from your R/C transmitter while flying.

  • LED Controller Updater

    A cross-platform GUI app to help update the firmware of the LED Controllers.

  • Dotfiles

    My personal collection of dotfiles (configuration files) for Linux installs.

  • Advent of Code solutions

    My solutions to the Advent of Code programming challenges

  • Google Calendar Overview

    A simple, single-file interface to get a 6-month view of a shared Google calendar, suitable for displaying on a TV.
